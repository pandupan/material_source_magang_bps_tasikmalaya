# Material Source Magang BPS Tasikmalaya 📊

> **Dataset hasil Kerangka Sampel Area (KSA) untuk pemantauan produksi padi — Kabupaten Tasikmalaya 2025**

Repositori ini berisi kumpulan dataset **Kerangka Sampel Area (KSA)** yang digunakan selama kegiatan magang di **BPS (Badan Pusat Statistik) Kabupaten Tasikmalaya**. Data ini mencatat hasil pengamatan lapangan titik-titik KSA untuk estimasi produksi padi.

Dataset tersedia dalam 9 versi (*v1* hingga *v9*) yang merefleksikan pembaruan dan validasi data secara bertahap selama periode magang.

---

## 📁 Dataset

| File | Deskripsi |
|------|-----------|
| `dataset_ksa_tasik_2025.xlsx` | Dataset utama KSA Tasikmalaya 2025 |
| `dataset_ksa_tasik_2025_v1.xlsx` hingga `v9.xlsx` | Dataset dengan pembaruan bertahap (9 versi) |

---

## 📋 Informasi Dataset

- **Sumber:** BPS Kabupaten Tasikmalaya
- **Tahun:** 2025
- **Format:** Microsoft Excel (.xlsx)
- **Metodologi:** Kerangka Sampel Area (KSA) — metode pengukuran produksi padi berbasis citra satelit dan validasi lapangan
- **Lokasi:** Kabupaten Tasikmalaya, Jawa Barat

---

## 🔍 Penggunaan Dataset

Dataset ini dapat digunakan untuk:
- Analisis produksi padi di wilayah Tasikmalaya
- Validasi data pertanian berbasis KSA
- Pembelajaran metodologi survei pertanian BPS
- Penelitian terkait ketahanan pangan dan produktivitas pertanian

## 🛠️ Tools yang Digunakan

- **Microsoft Excel** / **Google Sheets** — Pembacaan dan analisis data
- **Python** (pandas, openpyxl) — Pemrosesan data programatik
- **QGIS / ArcGIS** — Analisis spasial KSA
