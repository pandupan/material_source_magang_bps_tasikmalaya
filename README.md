# 📊 Material Sumber Magang BPS Tasikmalaya

Repositori ini berisi **dataset dan bahan** yang digunakan selama kegiatan magang di **BPS (Badan Pusat Statistik) Kabupaten Tasikmalaya**.

## 📋 Dataset

Berisi file Excel (.xlsx) yang merupakan data **Ketahanan Sosial Area (KSA)** untuk wilayah Tasikmalaya:

| File | Keterangan |
|------|-----------|
| `dataset_ksa_tasik_2025.xlsx` | Dataset utama KSA Tasikmalaya 2025 |
| `dataset_ksa_tasik_2025_v1.xlsx` — `v9.xlsx` | Dataset revisi/update |

## 🎯 Tujuan

Dataset ini digunakan untuk:
- Analisis ketahanan sosial area di Kabupaten Tasikmalaya
- Prediksi dan visualisasi data produksi pertanian
- Pengembangan aplikasi monitoring KSA

## 📄 Lisensi

Data ini bersumber dari BPS Tasikmalaya dan digunakan untuk kepentingan akademis & penelitian.
